Mini-labo VMware avec firewall

Réseau virtuel isolé + filtrage + tests de communication

Description

Création d’un réseau isolé contenant deux machines virtuelles reliées à un firewall. Objectif : tester des règles de filtrage et observer les communications.

Objectifs techniques

  • Gestion de plusieurs VMs
  • Création d’un réseau interne isolé
  • Filtrage simple via firewall
  • Tests de connectivité (ping, ports…)
